# Sakura Agent 启动指南

## 环境要求

- JDK 21
- Maven 3.8+
- Redis 7.x（Windows 版）
- Docker Desktop（用于运行 Milvus）

## 一、启动依赖服务

### 1. 启动 Redis

Windows 上 Redis 一般装在某个目录下，打开终端进入 Redis 目录启动：

```powershell
# 进入你的 Redis 安装目录，例如：
cd C:\Redis

# 启动 Redis 服务端
redis-server.exe
```

启动后终端会显示 Redis 的 logo 和 `Ready to accept connections`，保持窗口不要关。

默认连接 `127.0.0.1:6379`，无需密码。如果填写过密码，在项目 `application.yml` 的 `spring.data.redis.password` 和 `redisson.single-server-config.password` 配置。

### 2. 启动 Milvus

先打开 Docker Desktop 桌面应用，等左下角状态变绿。

```powershell
# 进入你放 docker-compose.yml 的目录
cd E:\tools\milvus

# 后台启动
docker compose up -d
```

默认连接 `localhost:19530`，无认证。

验证是否启动成功：

```powershell
docker compose ps
# STATUS 列应显示 Up 或 healthy
```

## 二、配置 API Key

编辑 `src/main/resources/application.yml`，替换 DashScope API Key：

```yaml
spring:
  ai:
    dashscope:
      api-key: 你的阿里云百炼API密钥
```

获取方式：[阿里云百炼控制台](https://bailian.console.aliyun.com/) → 获取 API Key。

---

## 三、启动项目

```powershell
# 1. 进入项目目录
cd sakura-agent-master

# 2. 编译（首次或改了代码后执行）
mvn clean compile -DskipTests

# 3. 启动
mvn spring-boot:run
```

启动成功后访问：
- **Swagger 接口文档**：http://localhost:8123/api/swagger-ui.html

---

## 四、首次初始化

项目首次启动时会自动：
1. 连接 Milvus，创建向量集合（`initialize-schema: true`）
2. 加载 `classpath:document/*.md` 中的绘梨衣角色知识文档
3. 语义分块 + LLM 关键词提取 → 分批写入向量库

启动日志中看到 `[Milvus RAG] 数据成功写入向量数据库！` 即初始化完成。

## 五、常见问题

### Q1: `Unable to connect to Redis server`

Redis 没启动。检查 `redis-server.exe` 打开的终端窗口是否还在运行。

### Q2: `ERR Client sent AUTH, but no password is set`

本机 Redis 没设密码，但项目 yml 里 `password:` 字段留了空。删掉 `password:` 那一整行（不是留空，是删除）。

### Q3: `DEADLINE_EXCEEDED` — Milvus 连不上

Milvus 没启动。检查两步：
1. Docker Desktop 桌面应用是否在运行（任务栏图标）
2. `docker compose ps` 检查 milvus 容器状态

没有 Milvus 可用的替代方案——在启动类排除：

```java
// SakuraAgentApplication.java 或 YuAiAgentApplication.java
@SpringBootApplication(exclude = {
    DataSourceAutoConfiguration.class,
    MilvusVectorStoreAutoConfiguration.class   // 临时跳过
})
```

### Q4: 启动类找不到 / 编译报错

先执行 `mvn clean compile`，确保 `target/classes` 已生成，再启动。

## 六、端口一览

| 服务 | 端口 | 用途 |
|------|------|------|
| Sakura Agent | 8123 | 主应用 |
| Redis | 6379 | 短期记忆 / 分布式锁 |
| Milvus | 19530 | 向量库（长期记忆 / RAG） |
| MCP Server（可选） | 8127 | 图片搜索工具服务 |
